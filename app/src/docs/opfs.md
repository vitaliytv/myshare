## Огляд

Цей модуль забезпечує зв'язок із кореневим каталогом системи файлів (OPFS) у браузерному середовищі, дозволяючи отримати доступ до постійного сховища об'єктів файлової системи. Функція, що повертає кореневий каталог OPFS, працює в режимі лише для читання. У разі виникнення будь-яких помилок, внутрішній механізм обробки помилок (fail-safe) гарантує стабільність роботи, повертаючи `null` замість викидання винятків.

## Поведінка

1. Викликаючи `opfsRoot`, система намагається отримати доступ до кореневого каталогу об'єкту File System (OPFS) у браузері.
2. Якщо доступ до OPFS успішний, функція повертає об'єкт, що представляє цей кореневий каталог.
3. У разі будь-якої помилки під час спроби доступу або якщо API недоступний, функція повертає `null`.

## Гарантії поведінки

- Read-only: не виконує операцій запису (ФС/БД).
- Перехоплює помилки і не пропускає винятків назовні (fail-safe).
- За певних помилок повертає порожнє значення (напр. `null`) замість винятку.
