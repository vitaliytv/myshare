## ADR: Ollama для перекладу EN→UA субтитрів

## Context and Problem Statement
Застосунок myshare отримує субтитри YouTube через supadata API. Коли українські субтитри відсутні, але є англійські, потрібно перекласти їх в українську мову та зберегти результат для повторного використання.

## Considered Options
* Ollama (локальний LLM через HTTP на `localhost:11434`)
* Інші варіанти в transcript не обговорювалися.

## Decision Outcome
Chosen option: "Ollama (локальний LLM)", because користувач прямо запросив Ollama; виявлено локально встановлену модель `gemma4:e4b`; HTTP-виклик через наявний `tauri-plugin-http` без нового Rust-коду.

### Consequences
* Good, because переклад відбувається локально без зовнішніх API-ключів або витрат.
* Bad, because функціонал доступний тільки на desktop/Mac (де запущено Ollama); на Android кнопка не відображається.

## More Information
- `app/src/ollama.js` — Ollama-клієнт, `POST http://localhost:11434/api/chat`, `stream: false`, `temperature: 0.2`
- `app/src/translation-cache.js` — кеш у `localStorage` (`translation_cache_v1`)
- Модель: `gemma4:e4b` (знайдена в `~/.ollama/models/`)
- Chunking: ~800 символів по `\n`-сепараторах
