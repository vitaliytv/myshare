# Ollama для перекладу EN→UA субтитрів (desktop-only)

**Status:** Accepted
**Date:** 2026-06-02

## Context and Problem Statement

Застосунок `myshare` отримує субтитри YouTube через supadata API. Коли українські субтитри відсутні, але є англійські, потрібно перекласти їх в українську мову та зберегти результат для повторного використання без зовнішніх API-ключів або хмарних витрат.

## Considered Options

- Ollama (локальний LLM через HTTP на `localhost:11434`)
- Інші варіанти в transcript не обговорювалися.

## Decision Outcome

Chosen option: "Ollama (локальний LLM)", because користувач прямо запросив Ollama; виявлено локально встановлену модель `gemma4:e4b`; HTTP-виклик через наявний `tauri-plugin-http` без нового Rust-коду; переклад відбувається локально без зовнішніх API-ключів.

### Consequences

- Good, because переклад відбувається локально без зовнішніх API-ключів або витрат; повторне відкриття перекладеного відео відкриває результат миттєво з кешу без повторного запуску Ollama.
- Bad, because функціонал доступний тільки на desktop/Mac (де запущено Ollama); на Android кнопка «Перекласти» не відображається (`!isAndroidPlatform()`).

## More Information

- `app/src/ollama.js` — `translateToUkrainian({onProgress})`, `chunkText` (чанки ≤800 символів по `\n`-сепараторах), `resolveModel` (дефолт `DEFAULT_MODEL = 'gemma4:e4b'`, fallback — перша з `/api/tags`); `POST http://localhost:11434/api/chat`, `stream: false`, `temperature: 0.2`
- Системний промпт: `"You are a professional EN→UK subtitle translator. Translate the user text into natural Ukrainian..."`
- `app/src/ollama.test.js` — тести Ollama-клієнта
- `app/src/translation-cache.js` — кеш перекладів у `localStorage` (ключ `translation_cache_v1`); `loadTranslations` / `saveTranslations`; запис містить `{ model, originalLang, segments: [{original, translated}] }`; `isValidEntry` відкидає биті записи; інвалідації кешу та TTL немає
- `app/src/translation-cache.test.js` — тести кешу
- UI: `app/src/App.vue` — прогрес-бар «Перекладаємо…» + відсоток; діалог порівняння два стовпці EN ↔ UA; кнопка «Перекласти» прихована на Android
- Tauri capability: `app/src-tauri/capabilities/default.json` — додані `{ "url": "http://localhost:11434/**" }` та `{ "url": "http://127.0.0.1:11434/**" }` (wildcard `http://**` не покривав нестандартні порти за логікою urlpattern в `tauri-plugin-http`; наявні записи `http://**` і `https://**` збережені)
- Тести: 67 JS (6 файлів) + 10 Rust — всі зелені

## Update 2026-06-02

Явне рішення щодо стратегії кешування: для обох типів кешу (мови відео — `caption-langs.js`, та переклади — `translation-cache.js`) обрано `localStorage` замість реактивного `ref` в пам'яті, бо `localStorage` переживає перезапуск застосунку і не потребує серверного сховища; `videoId` є стабільним ключем для обох типів кешу. Інвалідації та TTL для обох кешів немає — transcript не фіксує рішення щодо терміну дії записів.
