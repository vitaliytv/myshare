## ADR Вибір debug APK для особистого встановлення на телефон

## Context and Problem Statement
Потрібно зібрати Android-застосунок Tauri та встановити його на фізичний пристрій для особистого використання. Виникло питання, яку конфігурацію збірки обрати.

## Considered Options
* Debug APK (непідписаний, без keystore)
* Release APK (підписаний, з keystore)

## Decision Outcome
Chosen option: "Debug APK", because користувач підтвердив «для себе» — keystore і підпис не потрібні, установка через «невідомі джерела» достатня.

### Consequences
* Good, because transcript фіксує очікувану користь: збірка проходить без налаштування keystore, APK готовий одразу.
* Bad, because debug APK значно більший (775 MB universal debug проти очікуваного меншого release): transcript фіксує це як відкрите питання в кінці сесії.

## More Information
Команда збірки: `bun --cwd=app run tauri android build --apk --debug`
Результат: `app/src-tauri/gen/android/app/build/outputs/apk/universal/debug/app-universal-debug.apk`
Android-проєкт ініціалізований: `app/src-tauri/gen/android/`

## ADR Використання universal APK замість arch-specific у скрипті `android:install`

## Context and Problem Statement
Скрипт `android:install` у кореневому `package.json` посилався на `apk/arm64/debug/app-arm64-debug.apk`, але `tauri android build --apk --debug` фактично згенерував universal-варіант за іншим шляхом. Скрипт не знаходив файл після збірки.

## Considered Options
* Залишити arm64-specific шлях і додати прапор Tauri для примусової arm64-збірки
* Змінити шлях у скрипті на `universal/debug/app-universal-debug.apk`

## Decision Outcome
Chosen option: "Змінити шлях у скрипті на universal", because universal APK підходить для будь-якого телефону і саме його генерує Tauri за замовчуванням у поточній конфігурації.

### Consequences
* Good, because transcript фіксує очікувану користь: `bun run android:install` тепер знаходить файл і встановлює APK в одну команду.
* Bad, because universal APK містить `.so`-бібліотеки для всіх архітектур (`arm64-v8a`, `armeabi-v7a`, `x86`, `x86_64`), що збільшує розмір файлу — підтверджено в transcript (775 MB).

## More Information
Змінений файл: `package.json`, скрипт `android:install`.
Старий шлях: `app/src-tauri/gen/android/app/build/outputs/apk/arm64/debug/app-arm64-debug.apk`
Новий шлях: `app/src-tauri/gen/android/app/build/outputs/apk/universal/debug/app-universal-debug.apk`
Команда встановлення: `adb install -r <шлях до apk>`
