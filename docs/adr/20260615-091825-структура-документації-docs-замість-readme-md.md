## ADR Зберігання прийнятих URL як масив-черга в localStorage

## Context and Problem Statement
`myshare` приймає посилання через Android Share intent і показував лише одне поточне посилання без збереження між сесіями. Потрібно зберігати прийняті посилання у локальному сховищі для подальшого перегляду.

## Considered Options
* **Варіант 1:** Записувати лише останній URL у той самий ключ `myshare.sharedText`
* **Варіант 2:** Зберігати масив усіх прийнятих URL у новому ключі `myshare.sharedUrls` (найновіший — індекс 0)

## Decision Outcome
Chosen option: "Варіант 2 — масив усіх прийнятих URL у `myshare.sharedUrls`", because користувач явно відповів «2» на питання агента про два варіанти API.

### Consequences
* Good, because transcript фіксує очікувану користь: UI розширено з одного посилання на `q-list` усіх прийнятих URL; при перезапуску застосунку `myshare` список відновлюється з `localStorage`.
* Bad, because transcript не містить підтверджених негативних наслідків.

## More Information

## ADR Заміна Ollama на omlx (OpenAI-compatible MLX endpoint)

## Context and Problem Statement
Переклад субтитрів YouTube EN→UA в `myshare` (desktop) виконувався через локальний Ollama-сервер (`http://localhost:11434`, Ollama REST API). Проект `nitra/task` вже перейшов на omlx — OpenAI-compatible MLX-сервер на Apple Silicon. Потрібно привести `myshare` до того самого стеку.

## Considered Options
* Зберегти Ollama (`localhost:11434`, Ollama `/api/chat` API)
* Перейти на omlx (`localhost:8000`, OpenAI `/v1/chat/completions` API)
* Інші варіанти в transcript не обговорювалися.

## Decision Outcome
Chosen option: "Перейти на omlx (`localhost:8000`, OpenAI `/v1/chat/completions` API)", because користувач вказав на аналогічний міграційний патерн у суміжному проєкті `nitra/task` і попросив повторити його в `myshare`.

### Consequences
* Good, because transcript фіксує очікувану користь: єдиний API-контракт (OpenAI-compatible) між `myshare` і `nitra/task`; 99 vitest-тестів пройшли без змін після рефакторингу.
* Bad, because transcript не містить підтверджених негативних наслідків.

## More Information
`app/src/ollama.js` і `app/src/ollama.test.js` — **видалено**. Новий `app/src/omlx.js` — `translateToUkrainian`, `listOmlxModels`, `DEFAULT_MODEL`; endpoint `http://127.0.0.1:8000/v1/chat/completions`, список моделей через `GET /v1/models`. `app/src-tauri/capabilities/default.json` — дозволений URL оновлено з `localhost:11434/*` на `localhost:8000/*` і `127.0.0.1:8000/*`. `app/src/model-pref.js` — `STORAGE_KEY` оновлено з `myshare.ollamaModel` на `myshare.omlxModel`. Accepted ADR-файли в `docs/adr/` щодо вибору Ollama не змінювалися — вони є архівними записами попередніх рішень.
