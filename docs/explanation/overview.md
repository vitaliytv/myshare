# myshare — Overview

`myshare` — Android-застосунок для приймання посилань через стандартний Android Share механізм. Користувач у будь-якому Android-застосунку натискає **Share** для посилання, обирає `myshare` у системному share sheet — застосунок отримує переданий URL і відображає його на екрані.

Мінімальний результат першої версії `myshare` — коректно прийняти `text/plain` share intent з URL і відобразити цей URL у UI.
